# Фінансовий трекер студента (консольний чат-бот на Python)

## Мета проєкту
Розробити консольного чат-бота, який допомагає студенту вести облік особистих витрат,
працювати з бюджетом, отримувати залишок та формувати звіти за категоріями.

---

## Концепція та логіка роботи
Бот працює у режимі діалогу через консоль. Користувач вводить команди, бот виконує
відповідні дії (додавання витрати, встановлення бюджету, фільтрація, звіти тощо).

### Загальний алгоритм
1. При запуску бот читає дані з файлу `finance_data.json` (якщо файлу немає — створює
   початковий стан з бюджетом 0 і порожнім списком витрат).
2. Показує вітання та підказку команди `допомога`.
3. У циклі очікує команди користувача.
4. Для команд, що змінюють дані (бюджет/витрати), бот:
    - оновлює структуру даних у пам’яті,
    - автоматично зберігає її у файл,
    - перевіряє перевищення бюджету після додавання витрати.
5. Завершує роботу за командою `вийти`.

---

## Збереження даних у файл
Дані зберігаються у JSON-файлі `finance_data.json`.

### Структура даних (приклад)
```json
{
  "budget": 2500.0,
  "expenses": [
    {
      "amount": 120.5,
      "category": "Їжа",
      "date": "2026-02-27",
      "comment": "обід"
    }
  ]
}
